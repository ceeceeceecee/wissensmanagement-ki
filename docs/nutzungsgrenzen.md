# Nutzungsgrenzen — Was das System kann und nicht kann

## Was das System kann

### Informationsbereitstellung
- Fragen zu Dienstvorschriften, Verwaltungsvorschriften und internen Prozessen beantworten
- Dokumente durchsuchen und relevante Abschnitte finden
- Quellenangaben mit Dokumentnamen und Abschnitten liefern
- Formulare und Merkblätter finden

### Effizienzsteigerung
- Suchzeiten für Informationen reduzieren
- Häufig gestellte Fragen schneller beantworten
- Neue Mitarbeitende bei der Einarbeitung unterstützen
- Wissenslücken identifizieren (unbeantwortete Anfragen)

### DSGVO-Konformität
- Alle Daten lokal verarbeiten
- Keine Datenübertragung an externe Dienste
- Rollenbasierte Zugriffskontrolle
- Vertrauliche Dokumente schützen

## Was das System NICHT kann

### Keine rechtsverbindlichen Entscheidungen
- Das System gibt KEINE rechtsverbindliche Auskünfte
- Es ersetzt KEINE behördlichen Bescheide oder Genehmigungen
- Es kann KEINE Rechtsberatung ersetzen
- Bei Rechtsfragen: Immer die Rechtsabteilung konsultieren

### Keine automatische Amtshandlungen
- Das System führt KEINE Amtshandlungen durch
- Es kann KEINE Anträge genehmigen oder ablehnen
- Es kann KEINE Verträge abschließen
- Es kann KEINE Gelder freigeben oder überweisen

### Keine finale Prüfung
- Alle Antworten müssen von Fachkräften geprüft werden
- Die KI kann fehlerhafte oder unvollständige Antworten liefern
- Quellen müssen immer überprüft werden
- Die menschliche Prüfung ist obligatorisch, nicht optional

### Einschränkungen der KI
- Die KI versteht keine Dokumente, die nicht im System sind
- Die KI hat kein aktuelles Wissen (nur hochgeladene Dokumente)
- Die KI kann Halluzinationen erzeugen (unbegründete Aussagen)
- Die Konfidenzbewertung ist ein Richtwert, keine Garantie

## Warnhinweise

### Rote Linien — Niemals
- Rechtsverbindliche Auskünfte erteilen
- Bescheide oder Genehmigungen erstellen
- Verträge abschließen
- Personenbezogene Daten verarbeiten (Name, Adresse, etc.)
- Strafverfahren oder Ermittlungen betreffen

### Gelbe Linien — Nur mit Prüfung
- Finanz- und Haushaltsfragen
- Personalangelegenheiten
- Vergabeverfahren
- Bau- und Planungsfragen
- Umwelt- und Naturschutzfragen

### Grüne Linien — Unbedenklich
- Allgemeine Prozessbeschreibungen
- Formular- und Merkblatt-Suche
- Öffnungszeiten und Kontaktinformationen
- Zuständigkeitsfragen
- Allgemeine Verfahrensbeschreibungen

## Haftungsausschluss

Dieses System dient ausschließlich der Informationsbereitstellung und
Wissensorganisation. Es übernimmt keine Gewähr für die Richtigkeit,
Vollständigkeit oder Aktualität der bereitgestellten Informationen.

Alle Entscheidungen, die auf Basis der bereitgestellten Informationen
getroffen werden, liegen in der alleinigen Verantwortung der handelnden
Person. Es wird ausdrücklich darauf hingewiesen, dass eine fachliche
Prüfung durch zuständige Fachkräfte erforderlich ist.
